Para rodar aprimeira vez e necessario estar no python 3.11

Para baixar por terminal: winget install Python.Python.3.11

Depois roda : Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
1. Apagar o ambiente virtual antigo

No PowerShell:

deactivate

Depois:

rm -r -fo .venv
🐍 2. Criar novo ambiente com Python 3.11
py -3.11 -m venv .venv

Ativar:

.venv\Scripts\activate
📦 3. Atualizar pip
python -m pip install --upgrade pip
📚 4. Instalar dependências

Se tiver requirements.txt:

pip install -r requirements.txt

Se não:

pip install mediapipe==0.10.9 opencv-python pandas numpy tqdm scikit-learn matplotlib flask tensorflow
🧪 5. Testar se o MediaPipe está correto
python -c "import mediapipe as mp; print(mp.__version__); print(hasattr(mp, 'solutions'))"

Deve retornar:

0.10.9
True
🎥 6. Organizar os vídeos

Coloque os vídeos em:

videos/data/

Exemplo:

Oi_Articulador1.mp4
Ajudar_Articulador1.mp4
Obrigado_Articulador1.mp4
Casa_Articulador1.mp4
Nada_Articulador1.mp4
⚙️ 7. Rodar o pré-processamento
python etapa2_preprocessamento.py

Vai gerar:

landmarks.csv
🧠 8. Treinar o modelo
python etapa3_treinamento.py

Vai gerar:

modelo_libras.keras
label_encoder.pkl
🚀 9. Rodar o sistema
python app.py
🌐 10. Abrir no navegador
http://localhost:5000
⚠️ 11. Regra importante

Sempre que adicionar novos vídeos:

python etapa2_preprocessamento.py
python etapa3_treinamento.py
python app.py
